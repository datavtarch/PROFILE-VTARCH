# UI Guidelines

## Product Feel

The app should feel like a focused personal productivity tool: clean, fast, mobile-friendly, and easy to scan.

## First Screens

- Auth screen: simple sign in/sign up flow.
- Main task screen: today, upcoming, overdue, completed.
- Task editor: title, notes, due date, priority, status.
- Settings: timezone and Telegram linking placeholder.

## Interface Rules

- Prioritize mobile layout first.
- Use clear status labels and compact controls.
- Avoid decorative landing-page sections.
- Keep repeated task items as simple cards or rows.
- Make primary actions easy to reach on mobile.
- Ensure text does not overflow buttons, cards, or navigation.

## Visual Direction

- Use a restrained, professional palette.
- Avoid a one-color-only theme.
- Use icons for common actions such as add, edit, complete, delete, filter, and settings.
